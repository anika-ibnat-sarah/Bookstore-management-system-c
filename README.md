# 📚 Bookstore Management System (C)
A simple Bookstore Management System built using C programming with arrays and menu-driven interface.

## 📌 Overview
This is a Bookstore Management System built in C programming.  
The system has **two panels**: **Staff** and **Customer**, implemented using arrays and functions in a menu-driven interface.

---

## 🚀 Features

### Staff Panel
- ➕ Add new books  
- ✏️ Update book information  
- ❌ Delete book records  
- 📖 View all books  

### Customer Panel
- 📖 View available books  
- 🔍 Search books by title or author  
- 🛒 Place an order  
- ❌ Cancel an order  

---

## 🛠️ Technologies Used
- C Programming  
- Arrays  
- Functions  
- Loops & Conditional Statements  

---

## 💡 How It Works
- Staff and Customer have separate menus.  
- Staff can manage the inventory (add, update, delete books).  
- Customers can browse, search, place, and cancel orders.  
- All data is stored in arrays during program execution.
